# Ota-sence

Privacy-first indoor monitoring system using RF sensing and AI.

## Vision

Ota-sence is an indoor monitoring platform designed to provide
real-time occupancy, activity, and room awareness without using cameras
or microphones.

The system aims to provide:

- Privacy-preserving monitoring
- Human occupancy detection
- Room mapping
- Activity recognition
- Multi-room support
- AI-powered alerts

## Hardware

- mmWave Radar
- ESP32-S3 Sensor Nodes
- Central AI Hub
